# Environment Configuration

Use `.env.example` files in `backend/` and `mobile/`.

Do not commit real secrets.

## Backend `.env` notable values
- `APP_NAME`
- `APP_KEY`
- `DB_CONNECTION`, `DB_HOST`, `DB_DATABASE`, `DB_USERNAME`, `DB_PASSWORD`
- `REDIS_HOST`

## Mobile `.env` notable values
- `APP_NAME`
- `API_BASE_URL`
