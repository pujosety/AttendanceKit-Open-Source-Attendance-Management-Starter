# FAQ

**Q:** Does AttendanceKit copy Hadirku source code?  
**A:** No. AttendanceKit is a generic, clean scaffold with new generated code and no legacy branding.

**Q:** Can I use it for production?  
**A:** It is structured like a starter kit. You should complete validation, testing, and hardening before production use.

**Q:** What are the demo credentials?  
**A:** Not provided by default. Seeders include a sample admin format in docs.

**Q:** Does it support offline mode?  
**A:** Yes, offline queue architecture is prepared in `mobile/lib/core/utils/offline_queue.dart`.
