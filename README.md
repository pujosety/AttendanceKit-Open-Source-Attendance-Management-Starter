## Features

### Mobile
- Login
- Dashboard
- Check In
- Check Out
- Daily Report
- Attendance History
- Leave Request
- Calendar
- Notifications
- Profile
- Settings

### Attendance Features
- GPS Location
- Camera Verification
- Photo Upload
- Offline Queue
- API Synchronization
